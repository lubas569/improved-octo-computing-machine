## Description
A python package to JIT and run ONNX model as Halide pipeline.

## Dependinces
- ONNX source code.
- Pybind11.
- libprotobuf.

## Usage:
```
git clone https://github.com/onnx/onnx.git 
export ONNX_SRC_DIR=
make model_cpp=
```

To use the package add onnx/app directory to your `PYTHONPATH`
